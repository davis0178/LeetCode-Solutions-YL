```python
class Solution:
    def rotate(self, matrix: List[List[int]]) -> None:
        """
        Do not return anything, modify matrix in-place instead.
        """
        m, n = len(matrix), len(matrix[0])

        for i in range(n):
            for j in range(i): # 主对角线下方元素
                matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]
        
        for row in matrix:
            row.reverse()
```

顺时针旋转 90 度等价于按照左上到右下的对角线翻转再水平翻转（两次翻转）
抽象为 general concept: ==**一次旋转等于两次翻转**==

如何实现**主对角线翻转**？遍历主对角线下方元素，与对应的（关于主对角线）对称的元素交换

如何实现**水平翻转**？row.reverse()
