```python
MAPPING = "", "", "abc", "def", "ghi", "jkl", "mno", "pqrs", "tuv", "wxyz"

class Solution:
    def letterCombinations(self, digits: str) -> List[str]:
        n = len(digits)
        ans = []
        path = [0] * n

        def dfs(i: int) -> None:
            if i == n:
                ans.append("".join(path))
                return
            
            digit = int(digits[i])
            for c in MAPPING[digit]:
                path[i] = c
                dfs(i + 1)
        
        dfs(0)
        return ans
```

`dfs(i)` 不但在枚举 i，还包括了之后的递归调用，因此包括后续部分的处理。
这里是定长 path，遍历当前拨号数字（`digit`）对应的所有字母（`MAPPING[digit]`），加入 path 中，并继续处理下一位 path
