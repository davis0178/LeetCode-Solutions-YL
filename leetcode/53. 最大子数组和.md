```python
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        s = accumulate(nums)
        min_s = 0
        ans = -inf

        for x in s:
            ans = max(ans, x - min_s)
            min_s = min(min_s, x)

        return ans
```

前缀和 + 贪心：维护前缀和的最小值，当前前缀和减去前缀和的最小值，就是当前元素结尾的子数组和的最大值。
min_s 应该初始化为 0 而不是 inf，因为可以不减去任何前缀和

2025.12.27: 这里不 initial=0 的话就要初始化 min_s 为 0，inital=0 的话就必须初始化 min_s 为 inf，因为 initial 的 0 不能被当做一个前缀和来计算答案。本质上是一件事情，那就是**初始化最小前缀和为 0**
