二分查找三种写法：
**1. 闭区间写法**
```python
class Solution:
    def searchInsert(self, nums: List[int], target: int) -> int:
        left, right = 0, len(nums) - 1
        while left <= right: # 小于等于（都是区间不为空）
            mid = (left + right) // 2
            if nums[mid] < target:
                left = mid + 1 # 手动加减
            else:
                right = mid - 1
        return left # return left，结束时右跑到左的左边，左指向答案
```

**2. 左闭右开区间写法：**
```python
class Solution:
    def searchInsert(self, nums: List[int], target: int) -> int:
        left, right = 0, len(nums)
        while left < right: # 小于
            mid = (left + right) // 2
            if nums[mid] < target:
                left = mid + 1
            else:
                right = mid # 开区间不用加减 1
        return left # 或者 right 也行，结束时左右重合，都指向答案
```

**3. 开区间写法：**
```python
class Solution:
    def searchInsert(self, nums: List[int], target: int) -> int:
        left, right = -1, len(nums) 
        while left + 1 < right: # 两个开区间之间要有 1
            mid = (left + right) // 2
            if nums[mid] < target:
                left = mid
            else:
                right = mid
        return right # return right 而不是 left，结束时右保持在左的右边，右指向答案
```

注意：找到 mid 后判断语句是==单 <== 然后更新 left，不能 <=
