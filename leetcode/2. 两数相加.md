```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def addTwoNumbers(self, l1: Optional[ListNode], l2: Optional[ListNode], carry = 0) -> Optional[ListNode]:
        if l1 is None and l2 is None and carry == 0:
            return None
        
        s = carry
        if l1:
            s += l1.val
            l1 = l1.next
        if l2:
            s += l2.val
            l2 = l2.next
        
        return ListNode(s % 10, self.addTwoNumbers(l1, l2, s // 10))
```

这道题链表顺序是个位到十位到百位。因此可以使用递归的方法，下一个要计算的内容，是和原问题相似但规模更小的子问题。

当前节点根据进位值（carry）、l1.val 和 l2.val 求出总和
用节点储存当前结果。**当前数位是总和整除 10**
递归处理下一位的结果。下一位的**进位值是总和取余 10**
