```python
DIRS = (0, 1), (1, 0), (0, -1), (-1, 0)
class Solution:
    def spiralOrder(self, matrix: List[List[int]]) -> List[int]:
        m, n = len(matrix), len(matrix[0])
        ans = []
        i = j = di = 0
        for _ in range(m * n): # 一共走 mn 步
            ans.append(matrix[i][j])
            matrix[i][j] = None # 记得标记已访问过
            x, y = i + DIRS[di][0], j + DIRS[di][1]
            if x < 0 or x >= m or y < 0 or y >= n or matrix[x][y] is None: # 出界或已访问过
                di = (di + 1) % 4
            i += DIRS[di][0]
            j += DIRS[di][1]
        return ans
```

**模拟转向**：
构造方向需要是**右下左上**的排序，这样才能用 di + 1 旋转方向

先用 x y **试一试**当前方向，如果出界或者已访问过则转向
这里 di 加一后一定要**模 4**
再真正更新 i j
