```python
class Solution:
    def minTimeToVisitAllPoints(self, points: List[List[int]]) -> int:
        ans = 0
        
        # 遍历相邻点使用 pairwise，每次都计算切比雪夫距离
        # 使用元组中两个元素将 x1 x2 y1 y2 精准赋值以便接下来的计算
        for (x1, y1), (x2, y2) in pairwise(points):
            ans += max(abs(x1 - x2), abs(y1 - y2))
        return ans
```
从点一到点二的水平距离为 $d_x$，垂直距离为 $d_y$，当存在 both 水平距离和垂直距离时，先使用对角线移动可以让两者都减少1。因此相当于先同时消耗水平距离和垂直距离，其中一个消耗完之后剩下的再单独走。所以这道题移动的策略其实就是移动水平和垂直距离之间更大的那个对应的步数。这个距离其实就是：两点之间的切比雪夫距离

**切比雪夫距离**：$max(d_x, d_y) = max(|x_1 - x_2|, |y_1 - y_2|)$

`pairwise` 的使用：
```python
nums = [10, 20, 15, 18]
for x, y in pairwise(nums):
	print(x, y)
# 10, 20
# 20, 15
# 15, 18
```
