```python
class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        m, n = len(matrix), len(matrix[0])
        i, j = 0, n - 1 # 找到右上角元素

        while i < m and j >= 0:
            if matrix[i][j] < target:
                i += 1
            elif matrix[i][j] > target:
                j -= 1
            else:
                return True
        
        return False
```

只要**矩阵的行列满足相同的单调性**，那么抓住右上角元素就可以进行排除法查找。将 O(mn) 的暴力查找时间复杂度优化为 O(m + n)

通过比较 target 和右上角，排除右上角对应的行或列，最终找到答案，或在没有候选元素后返回 False
