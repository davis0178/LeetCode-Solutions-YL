```python
class Solution:
    def sortedSquares(self, nums: List[int]) -> List[int]:
        n = len(nums)
        ans = [0] * n
        left, right = 0, n - 1
        pos = n - 1

        while left <= right:
            if abs(nums[left]) > abs(nums[right]):
                ans[pos] = nums[left] ** 2
                left += 1
            else:
                ans[pos] = nums[right] ** 2
                right -= 1
            pos -= 1
    
        return ans
```

如果用双指针 left = 0 right = n - 1，那么最大平方一定来自于两端
每一步只要比较两端绝对值，就能确定当前最大平方
用指针 p 指向答案末端，从后往前填

用 nums[left] * nums[left] 会更快
