```python
class Solution:
    def bestClosingTime(self, customers: str) -> int:
        min_penalty = penalty = ans = 0
        for i, c in enumerate(customers):
            penalty += 1 if c == "N" else -1
            if penalty < min_penalty:
                min_penalty = penalty
                ans = i + 1
        return ans
        
```

如果时间 0 就关门，代价是数组中 Y 的数量
这里要理解一个概念，如果要比较物品的温度，**基准**是华氏度还是摄氏度都可以
因此没有必要统计开始前 Y 的数量，直接在基准 0 上比较所有关门时间

当关门时间右移时，如果新包括进来的时间段有人来（Y），cost 就减少 1。如果新时间段无人来（Y），cost 就增加 1。储存 penalty 最小值，更新 ans
