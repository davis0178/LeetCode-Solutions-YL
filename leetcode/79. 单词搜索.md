```python
class Solution:
    def exist(self, board: List[List[str]], word: str) -> bool:
        m, n = len(board), len(board[0])

        def dfs(i: int, j: int, k: int) -> bool:
            if board[i][j] != word[k]:
                return False
            if k == len(word) - 1:
                return True
            board[i][j] = ''
            for x, y in (i + 1, j), (i - 1, j), (i, j + 1), (i, j - 1):
                if 0 <= x < m and 0 <= y < n and dfs(x, y, k + 1):
                    return True
            board[i][j] = word[k]
            return False
        return any(dfs(i, j, 0) for i in range(m) for j in range(n))
```
我们需要知道当前匹配到 word 第几个字母了，所以加一参数 k 处理 word
匹配失败，return False
匹配成功且是最后一个字母，return True
递归过程中，先直接修改矩阵把字母置为空，返回 False 前再恢复为原来的值。（因为已经匹配成功，所以当前被置空的值就是 word[k]）
