**方法一：前序遍历**
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def isValidBST(self, root: Optional[TreeNode], left=-inf, right=inf) -> bool:
        if root is None:
            return True
        x = root.val
        return left < x < right and self.isValidBST(root.left, left, x) and self.isValidBST(root.right, x, right)
```
dfs 额外传入两个参数，表示根节点到当前路径上的最小值和最大值
每次都比较确定当前节点的值在最小值和最大值之间（往左走，值要小于 x，往右走，值要大于 x）

**方法二：中序遍历**
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    pre = -inf # 前一个元素的值
    def isValidBST(self, root: Optional[TreeNode]) -> bool:
        if root is None:
            return True
        
        # 中序遍历，先处理左子树
        if not self.isValidBST(root.left):
            return False
        
        # 再处理根节点
        if root.val <= self.pre: # 需要大于前一个元素的值
            return False
        self.pre = root.val # 更新前一个元素的值
        
        # 最后处理右子树
        return self.isValidBST(root.right)
```
二叉搜索树自然想到中序遍历，而中序遍历时可以把二叉搜索树看作有序数组
遍历有序数组时，通过比较相邻元素的大小就可以验证这是有序数组

**方法三：后序遍历**
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def isValidBST(self, root: Optional[TreeNode]) -> bool:
        def dfs(node: Optional[TreeNode]) -> Tuple:
            if node is None:
                return inf, -inf
            l_min, l_max = dfs(node.left)
            r_min, r_max = dfs(node.right)
            x = node.val
            if x <= l_max or x >= r_min:
                return -inf, inf
            return min(l_min, x), max(r_max, x)
        return dfs(root)[1] != inf
```
