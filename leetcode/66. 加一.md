```python
class Solution:
    def plusOne(self, digits: List[int]) -> List[int]:
        for i in range(len(digits) - 1, -1, -1):
            if digits[i] < 9:
                digits[i] += 1
                return digits
            digits[i] = 0
        
        return [1] + digits
```

从右往左找第一个不等于 9 的数，这个第一个不等于 9 的数加一，之前的数全清零
如果所有数都是 9，那就在左端多加一个 [0]
