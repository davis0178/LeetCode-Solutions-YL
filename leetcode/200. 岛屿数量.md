```python
class Solution:
    def numIslands(self, grid: List[List[str]]) -> int:
        m, n = len(grid), len(grid[0])
        dirs = (1, 0), (0, 1), (-1, 0), (0, -1)

        def dfs(i: int, j: int) -> None:
            if i < 0 or i >= m or j < 0 or j >= n or grid[i][j] != '1':
                return
            grid[i][j] = '2'
            
            for d in dirs:
                dfs(i + d[0], j + d[1])
        
        ans = 0
        for i in range(m):
            for j in range(n):
                if grid[i][j] == '1':
                    ans += 1
                    dfs(i, j)
        
        return ans
```
最基本的图题：在访问过的格子插上旗子。
dfs 写法：边界条件是出界或者进到水里（也算一种出界），到达的格子插旗（赋值一个不为原始值的其他值），最后合法的方向上都尝试走一步
遍历方法：遍历到新宝岛，答案就加一，同时进行 dfs 占领整个岛屿
