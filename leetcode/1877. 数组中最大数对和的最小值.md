```python
class Solution:
    def minPairSum(self, nums: List[int]) -> int:
        n = len(nums)
        res = 0
        nums.sort()
        for i in range(n // 2):
            res = max(res, nums[i] + nums[n - 1 - i])
        return res
```

排序 + 贪心：对于 n 个数，由第 $k$ 大于第 $k$ 小组成的数对可以使最大数对和最小
除此以外任意的拆分方法得到的最大数对和一定大于等于这个拆分方法得到的最大数对和
遍历每一个用这种方法得到的数对，维护它们和的最大值
