```python
class Node:
    __slots__ = 'son', 'end'

    def __init__(self):
        self.son = {}
        self.end = False

class Trie:

    def __init__(self):
        self.root = Node()

    def insert(self, word: str) -> None:
        cur = self.root
        for c in word:
            if c not in cur.son:
                cur.son[c] = Node()
            cur = cur.son[c]
        cur.end = True
    
    def find(self, word: str) -> int:
        cur = self.root
        for c in word:
            if c not in cur.son:
                return 0
            cur = cur.son[c]
        return 1 if cur.end else 2

    def search(self, word: str) -> bool:
        return self.find(word) == 1
        

    def startsWith(self, prefix: str) -> bool:
        return self.find(prefix) != 0
```

记录单词我们可以用：**二十六叉树**，相同的路径（移动方向）代表相同的单词

初始化：创建一棵 26 叉树 root，每个节点包含儿子和一个布尔值 end，表示是否为终止节点
插入：遍历字符串 word，用 cur 表示在 26叉树中的位置。如果 word 不是当前儿子，就创建一个新儿子。更新 cur 为当前儿子。结束遍历时，把当前节点的 end 设置为 True
寻找：还是 word 和 cur，如果 word[i] 不是 cur 的儿子，返回 0（search 和 startsWith 都无法接受，返回 False）。遍历结束，cur 是 end 返回 2（精准匹配），cur 不是 end 返回 1（startswith）
