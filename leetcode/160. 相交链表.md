```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def getIntersectionNode(self, headA: ListNode, headB: ListNode) -> Optional[ListNode]:
        p, q = headA, headB
        while p is not q:
            p = p.next if p else headB
            q = q.next if q else headA
        return p
```

如何判断链表相不相交？
假设相交链表结尾多一个空节点，即使不相交也假设它们相交于空节点。
初始化指针 p q，不断循环 p q 各走一步直到 p q 相遇。当 p 或 q 走到空节点**再走一步**（p q 没有相遇）就传送回另一个链表的 head。这样当 p q 相遇时，相遇节点就是链表相交的节点（如果相遇在空节点，相当于两个链表相交于空节点，两个链表不相交）
